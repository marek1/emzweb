Frontend-Entwicklung in Berlin

Javascript ist unsere Programmiersprache

Wir erstellen Frontend-Applikationen basierend auf HTML, CSS und natürlich Javascript, der Programmiersprache des Internets. Applikationen können mit "Plain Vanilla" Javascript programmiert sein, oder aber auch mit Hilfe von Bibliotheken wie Jquery (wenn dies von der Größe her Sinn macht).

Desweiteren benutzen wir moderne MV* Framework, wie Backbone.js und/oder Angular.js, um den Javascript Code besser zu organisieren, modularisieren und vor allem skalierbar zu machen.

Liegt uns sehr am Herzen. Die Reduzierung von Server Requests, den Einsatz von Sprites, die Minifizierung von Code sind ebenso wichtig, wie der behutsame Einsatz von externen Bibliotheken. Deswegen haben wir auch bei dieser Seite auf den Einsatz von Frameworks und auf Bibliotheken, wie z.B. Jquery und Bootstrap verzichtet.
