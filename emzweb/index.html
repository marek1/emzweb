Frontend-Entwicklung in Berlin

Javascript ist meine Programmier-sprache : Ich erstelle Frontend-Applikationen basierend auf Backbone.js und/oder Angular.js.

Diese MV* Frameworks helfen, den Javascript Code zu separieren, modularisieren und organisieren.

... Code-Beispiele auf Github

HTML5 : ist meine Markup-Sprache

Markup, das auch Mobile-Browser versteht.

CSS3 : ist mein Style-Sprache

Header, nav, article, section, footer werden genauso eingesetzt wie media queries und transitions.

Performance !

Liegt mir sehr am Herzen. Die Reduzierung von Server Requests, den Einsatz von Sprites, die Minifizierung von Code sind ebenso wichtig, wie der behutsame Einsatz von externen Bibliotheken. Deswegen habe ich auch bei dieser Seite auf den Einsatz eines Frameworks, auf Jquery und auf Bootstrap verzichtet.
